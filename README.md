# Installation

Tools needed :
 * Docker
 * docker-compose

Clone the repository
```bash
    git clone https://github.com/aw09/Message-API
```

Build image
```
    docker-compose build
```

Run 
```docker
    docker-compose up
```


Check running container
```
    docker ps
```

Check API
```
    ipserver:2000/users
```
